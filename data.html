Data

  1. Create a new folder somewhere on your computer that’s easy to get to (e.g., your Desktop). Name it bioconnector. Inside that folder, make a folder called data, all lowercase.
  2. Download datasets as needed. Save these data files to the new bioconnector/data folder you just made. Click to download. If data displays in your browser, right-click and select Save link as… (or similar) to save to the desired location.
  • airway_metadata.csv
  • airway_rawcounts.csv
  • annotables_grch37.csv
  • annotables_grch38.csv
  • brauer2007_messy.csv
  • brauer2007_sysname2go.csv
  • brauer2007_tidy.csv
  • gapminder.csv
  • heartrate2dose.csv
